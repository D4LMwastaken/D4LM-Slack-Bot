# D4LM's Slack Bot
 My first Slack Bot!

## First Iteration
The first iteration follows the tutorial and is written in Node.js

## Second Iteration
The second iteration uses [Slack API for Go](https://github.com/slack-go/slack) and Gemini API.
### Command List
| Command Name    | Description                       |
|-----------------|-----------------------------------|
| /d4lm-ping      | Pings the bot!                    |
| /d4lm-help      | Shows you all the commands!       |
| /d4lm-catfact   | Replies with a cat fact!          |
| /d4lm-joke      | Replies with a joke!              |
| /d4lm-ask       | Replies with the answer you seek! |
| /d4lm-ascii-art | Generates ascii art               |
